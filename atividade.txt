Exercício 1

Variável: nota = 10
Sentença: nota <= 10

Resolução:
10 <= 10 → Verdadeiro

Resultado: Verdadeiro

 Exercício 2

Variáveis: nota = 6, faltas = 4
Sentença: (nota <= 6) and (faltas <= 3)

Resolução:
nota <= 6 → 6 <= 6 → Verdadeiro
faltas <= 3 → 4 <= 3 → Falso

Verdadeiro and Falso → Falso

Resultado: Falso

Exercício 3

Variáveis: convidados = 3, fumante = False
Sentença: (convidados > 4) or (fumante == True)

Resolução:
convidados > 4 → 3 > 4 → Falso
fumante == True → False == True → Falso

Falso or Falso → Falso

Resultado: Falso

Exercício 4

Variável: dia = "qua"
Sentença: (dia == "sab") or (dia == "dom")

Resolução:
"qua" == "sab" → Falso
"qua" == "dom" → Falso

Falso or Falso → Falso

Resultado: Falso

Exercício 5

Variável: feriado = True
Sentença: not (feriado == False)

Resolução:
feriado == False → True == False → Falso
not Falso → Verdadeiro

Resultado: Verdadeiro
